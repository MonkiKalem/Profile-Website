Portfolio

These are some of my project and mini design project

DailyScope - Mobile

DailyScope is an Android application developed using Flutter, designed as the final project for the Mobile Cloud Computing course. The app fetches real-time news updates from various sources through the integration of NewsAPI, providing users with a seamless and interactive way to stay informed on current events.

NgajarKuy

NgajarKuy is the final project for the Mobile Multimedia Solutions course at Bina Nusantara University. It is an iOS application developed using Xcode and Swift, designed as a B2B solution for school teachers. The app aims to enhance the teaching experience by providing a platform for teachers to manage their classes, interact with students, and streamline their administrative tasks.

Eco Eats

Eco Eats is a project developed for Software Engineering course at Binus University. This project is an Android application prototype designed to promote sustainable eating habits.
